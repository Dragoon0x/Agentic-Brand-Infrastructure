# Contributing

This is an open standard. It gets better when more people use it, break it, and improve it.

## How to Contribute

1. **Fork** the repo
2. **Create a branch** for your feature or fix
3. **Write tests** if applicable
4. **Submit a PR** with a clear description of what you changed and why

## What We Need

- Real-world testing and feedback
- Schema improvements based on edge cases
- New templates for different industries
- Integration examples with popular AI frameworks
- Documentation improvements
- Translations

## Code of Conduct

Be decent. Be direct. Build something useful. No ego, no gatekeeping.

## Questions?

Open an issue. Tag it with `question`.
